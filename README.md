# typeorm-multer-upload
Desafio 06: Banco de dados e upload de arquivos no Node.js com TypeORM e Multer

Nesse desafio, foi continuado o desenvolvendo da aplicação de gestão de transações, treinando o que foi aprendido 
até agora no Node.js junto ao TypeScript, mas dessa vez incluindo o uso de banco de dados com o TypeORM 
e envio de arquivos com o Multer!

Essa será uma aplicação que deve armazenar transações financeiras de entrada e saída 
e permitir o cadastro e a listagem dessas transações, além de permitir a criação de novos registros no banco de dados
a partir do envio de um arquivo csv.

## Tecnologias
- Node.js
- Typescript
- TypeORM
- Multer
